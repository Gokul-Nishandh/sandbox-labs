| Method   | Endpoint          | Description                                               | Example `curl`                                         |
| :------- | :---------------- | :-------------------------------------------------------- | :----------------------------------------------------- |
| **GET**  | `/nodes`          | List all nodes (reads from `inventory.json`)              | `curl -X GET http://localhost:5000/nodes`              |
| **POST** | `/nodes`          | Create a new node (creates overlay and adds to inventory) | `curl -X POST http://localhost:5000/nodes`             |
| **POST** | `/nodes/:id/run`  | Start the specified node VM (by name, e.g. `node_1`)      | `curl -X POST http://localhost:5000/nodes/node_1/run`  |
| **POST** | `/nodes/:id/stop` | Stop the specified node                                   | `curl -X POST http://localhost:5000/nodes/node_1/stop` |
| **POST** | `/nodes/:id/wipe` | Wipe (reset) the specified nodeâ€™s overlay                 | `curl -X POST http://localhost:5000/nodes/node_1/wipe` |
| **POST** | `/nodes/wipeAll`  | Stop and wipe **all** nodes in `inventory.json`           | `curl -X POST http://localhost:5000/nodes/wipeAll`     |

| Method   | Endpoint            | Description                                                  | Example `curl`                                        |
| :------- | :------------------ | :----------------------------------------------------------- | :---------------------------------------------------- |
| **POST** | `/nodes/router`     | Create the router overlay (router_1) and add it to inventory | `curl -X POST http://localhost:5000/nodes/router`     |
| **POST** | `/nodes/router/run` | Start the router VM and create a Guacamole connection        | `curl -X POST http://localhost:5000/nodes/router/run` |
